Gem Collector

A random number will be generated at the start of the game

There are four gems each with a random value assigned. By clicking on each specific crystal you will add that value to your total score.

You win when you match your score with the random number. You lose if your score is more than the random number.

The value of each gem is hidden until you click on it. Each time the game starts the game will change the value of the gems.
